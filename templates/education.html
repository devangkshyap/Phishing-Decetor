Phishing Awareness

What is Phishing?

Phishing is a cyber attack that uses disguised email as a weapon. The goal is to trick the email recipient into believing that the message is something they want or need.

Did you know?

91% of cyber attacks start with a phishing email, making it the most common attack vector.

Common Phishing Indicators

Urgency Tactics

Messages claiming "urgent action required" or "expires today"

Suspicious Senders

Emails from unknown senders or suspicious domains

Malicious Links

URLs that don't match the claimed destination

Poor Grammar

Spelling mistakes and grammatical errors

Phishing Email Examples

PHISHING EXAMPLE From: security@paypaI-verification.com
Subject: URGENT: Your PayPal Account Will Be Suspended

Dear Customer,

Your PayPal account has been limited due to suspicious activity. You must verify your account within 24 hours or it will be permanently suspended.

Click here to verify your account immediately: http://paypal-verification.secure-login.net

Red Flags:
  • Misspelled domain (paypaI instead of paypal)
  • Urgent language and threats
  • Suspicious URL that doesn't match PayPal
  • Generic greeting

URL Security

Anatomy of a Secure URL

https://www.example.com/secure/login
HTTPS

Secure, encrypted connection

Domain

Legitimate, recognizable domain name

Path

Logical page structure

Dangerous URL Patterns

http://192.168.1.100/login
IP Address instead of domain name
https://paypal.fake-security.tk
Suspicious subdomain + unusual TLD
https://bit.ly/abc123
URL shortener hiding real destination

Media File Security

File-Based Threats

Cybercriminals often disguise malicious software as innocent-looking media files. These can include:

Malware

Malicious code hidden in images or videos that can infect your system

Trojans

Programs that appear harmless but contain hidden malicious functionality

Spyware

Software that secretly monitors and collects personal information

Ransomware

Malicious software that encrypts files and demands payment

File Security Best Practices

Scan Before Opening

Always scan files with antivirus software before opening

Check File Extensions

Be suspicious of files with double extensions (e.g., image.jpg.exe)

Trust Your Sources

Only download files from trusted, verified sources

Keep Software Updated

Ensure your operating system and antivirus are up to date
